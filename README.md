#TaskFlow

🇩🇪 Deutsch

Eine in JavaScript entwickelte Task-Management-Webanwendung.
Die Anwendung nutzt dynamisches DOM-Rendering und LocalStorage, um Aufgaben mit verschiedenen Status zu verwalten und dauerhaft im Browser zu speichern.

Funktionen

• Aufgaben erstellen und löschen
• Aufgabenstatus verwalten (Nicht erledigt, In Bearbeitung, Erledigt)
• Persistente Speicherung mit LocalStorage
• Dynamische Aktualisierung der Benutzeroberfläche
• Dunkelmodus

Technologien

• HTML
• CSS
• JavaScript
• LocalStorage

Ziel des Projekts

Dieses Projekt wurde entwickelt, um praktische Erfahrung mit JavaScript, DOM-Manipulation und Browser-Speichertechnologien zu sammeln.
Der Fokus lag auf der Entwicklung einer einfachen, aber funktionalen Webanwendung zur Aufgabenverwaltung.

🇬🇧 English

A task management web application built with JavaScript.
The application uses dynamic DOM rendering and LocalStorage to manage tasks with different statuses and persist them in the browser.

Features

• Create and delete tasks
• Task status management (Not Done, In Progress, Done)
• Persistent storage using LocalStorage
• Dynamic user interface updates
• Dark mode

Technologies

• HTML
• CSS
• JavaScript
• LocalStorage

Project Goal

This project was created to gain practical experience with JavaScript, DOM manipulation, and browser storage technologies.
The goal was to build a simple but functional task management web application.
